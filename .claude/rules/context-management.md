# Context 管理

## Context 監控

| 使用率 | 動作 |
|--------|------|
| < 70% | 正常執行 |
| 70-80% | 精簡輸出（禁止完整程式碼展示、詳細日誌、重複說明） |
| 80-90% | 警告 + 保存進度 |
| > 90% | 保存 + 停止 + 建議新會話 |

## Memory Bank 按需載入

| 檔案 | 觸發條件 |
|------|---------|
| progress.yaml | 查詢/更新進度、開始/完成任務 |
| decisions.yaml | 做決策前、查詢過去決策 |
| preferences.yaml | 用戶表達偏好 |

## 記憶安全

- 記憶檔案禁止存放明文密碼、API key、token 等機密資訊
- IF 使用者提供密碼/token THEN 只記錄「存在哪個檔案」（如「密碼在 .env」），禁止記錄明文值
- 理由：記憶檔案可能存在 git repo 或本機檔案系統，明文機密有洩露風險（2026-04-01 回顧）

## 自動記錄

- 任務完成 → 更新 progress.yaml
- 錯誤解決 → 記錄解法
- 做出決策 → 記錄 decisions.yaml（決策、理由、影響）
