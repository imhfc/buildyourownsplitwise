# Context 管理

## Context 監控

| 使用率 | 動作 |
|--------|------|
| < 70% | 正常執行 |
| 70-80% | 精簡輸出（禁止完整程式碼展示、詳細日誌、重複說明） |
| 80-90% | 警告 + 保存進度 |
| > 90% | 保存 + 停止 + 建議新會話 |

## Memory Bank 按需載入

| 檔案 | 觸發條件 |
|------|---------|
| progress.yaml | 查詢/更新進度、開始/完成任務 |
| decisions.yaml | 做決策前、查詢過去決策 |
| preferences.yaml | 用戶表達偏好 |

## 自動記錄

- 任務完成 → 更新 progress.yaml
- 錯誤解決 → 記錄解法
- 做出決策 → 記錄 decisions.yaml（決策、理由、影響）
